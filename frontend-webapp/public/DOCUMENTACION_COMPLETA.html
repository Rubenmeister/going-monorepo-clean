1. Introducción

Visión General

Going es una plataforma integral de viajes que permite a los usuarios buscar y reservar transporte, alojamiento, tours y experiencias en un solo lugar.

Estado Actual

✓ Implementación Completada
Se han implementado todos los componentes principales con un sistema de traducción multiidioma profesional.

Stack Tecnológico

  • Next.js 15 (App Router)
  • React 19 con TypeScript
  • Tailwind CSS para estilos
  • Context API para estado global
  • Custom Hooks personalizados

2. Arquitectura General

Estructura de Carpetas

frontend-webapp/src/app/ ├── components/ │ ├── Sidebar.tsx ← Menú lateral │ ├── Navbar.tsx ← Barra superior │ ├── Footer.tsx ← Pie de página │ └── LanguageSwitcher.tsx ← Selector de idioma │ ├── contexts/ │ └── LanguageContext.tsx ← Contexto global │ ├── hooks/ │ └── useTranslation.ts ← Hook de traducción │ ├── utils/ │ └── translations.ts ← Diccionario │ ├── account/ │ └── page.tsx ← Página de cuenta │ ├── services/ ← Páginas de servicios │ ├── transport/page.tsx │ ├── tours/page.tsx │ ├── accommodation/page.tsx │ └── experiences/page.tsx │ ├── legal/ ← Páginas legales │ ├── terms/page.tsx │ ├── privacy/page.tsx │ ├── cookies/page.tsx │ └── contact/page.tsx │ └── layout.tsx ← Layout principal

Flujo de Datos

User selecciona idioma
↓
LanguageSwitcher actualiza LanguageContext
↓
Se guarda en localStorage
↓

↓

6. Guía de Uso

6.1 Iniciando el Servidor

# En la carpeta del proyecto cd /home/user/going-monorepo-clean # Instalar dependencias npm install # Iniciar servidor npm run dev:webapp # Abre en navegador http://localhost:4200

6.2 Cambiando de Idioma

  1. Busca el selector en la esquina superior derecha
  2. Haz click en [🇪🇸 Español ▼]
  3. Se abrirá un dropdown con 3 opciones
  4. Selecciona el idioma deseado
  5. ¡Toda la interfaz se traduce automáticamente!
  6. Tu selección se guarda automáticamente

6.3 Usando el Sidebar

En Mobile:
  • Haz click en el icono ☰ (esquina superior izquierda)
  • Se abrirá el sidebar lateral
  • Click en una opción para navegar
  • El sidebar se cierra automáticamente
  • O haz click afuera para cerrar
En Desktop:
  • El sidebar está siempre visible
  • Haz click en cualquier opción para navegar
  • El usuario y logout están visibles

6.4 Navegando el Sitio

Rutas principales:

• / - Home (búsqueda de viajes)
• /account - Mi Cuenta (4 tabs)
• /services/* - Servicios específicos
• /about - Información de empresa
• /contact - Formulario de contacto
• /legal/* - Documentos legales
