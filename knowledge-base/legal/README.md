# Legal Going

Documentos legales **vinculantes** que rigen la relación entre Going (Thorn AI
Technologies) y sus usuarios (pasajeras/pasajeros, conductoras/conductores,
empresas).

## Filosofía

A diferencia de `policies/` (políticas operativas — cómo funcionan las cosas
en el día a día), `legal/` contiene los textos legales que el usuario
**acepta** al usar Going.

**Cambios acá requieren:**
1. Revisión con un abogado ecuatoriano antes de publicar.
2. Notificación a usuarios existentes si el cambio es material.
3. Versionado: nunca borrar versiones anteriores, agregar fecha de vigencia.

## Mapa

```
legal/
├── README.md                ← este archivo
├── terminos.md              ← Términos y Condiciones de uso del servicio
├── privacidad.md            ← Política de Privacidad (cumple LOPDP Ecuador)
├── cookies.md               ← Política de cookies del sitio web
├── conductores.md           ← Términos específicos para conductoras/conductores
└── corporativo.md           ← Términos específicos para cuentas Going Empresas
```

## Normativa aplicable (Ecuador)

- **LOPDP** — Ley Orgánica de Protección de Datos Personales (2021)
- **Código del Comercio** y leyes de defensa del consumidor
- **Reglamento de Transporte Terrestre, Tránsito y Seguridad Vial**
- **Régimen tributario** del SRI (facturación electrónica)
- **Ordenanzas municipales** de cada cantón cubierto (donde aplique a
  transporte intra-cantonal)
