# Going Superapp - Design Guidelines

Official brand guidelines based on the Going PDF brand book.

---

## Logo Assets

| Variant | File Path | Usage |
|---------|-----------|-------|
| Color | `/assets/brand/going-logo-color.png` | Primary use on light backgrounds |
| Black | `/assets/brand/going-logo-black.png` | Monochrome print, dark on light |
| White | `/assets/brand/going-logo-white.png` | On dark/colored backgrounds |
| App Mockup | `/assets/brand/going-app-mockup.png` | Marketing materials |

### Logo Structure
- **Icon**: Stylized "G" with circular loop (represents movement/journey)
- **Wordmark**: "Going" in bold sans-serif (black text)
- **Primary Color**: Going Red `#F26155` (coral-red tone)

## Typography

| Use | Font | Weight |
|-----|------|--------|
| Headings | Space Grotesk | 900 (Black) |
| Body | Inter | 400-500 |
| Logo text | Custom sans-serif | Bold |

## Component Patterns

### Cards
- White background with `border: 2px solid black`
- Neo-brutalist shadow: `box-shadow: 4px 4px 0 black`
- Hover: `transform: translateY(-8px)`

### Buttons
| Type | Style |
|------|-------|
| Primary | Going Red bg, white text, rounded-full |
| Secondary | Black bg, white text |

---

## DO's and DON'Ts

✅ **DO**: Use official logo files, real photos, Going Red `#F26155`

❌ **DON'T**: Use drawings instead of real photos, alter logo colors
